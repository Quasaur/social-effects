# Project Brief: Social Effects

## Overview
Social Effects is a macOS CLI tool for generating professional-quality 15-30 second videos with stoic aesthetics for social media platforms (TikTok, Instagram Reels, YouTube Shorts).

## Core Purpose
Create cinematic wisdom videos that combine AI-generated background visuals, text-to-speech narration, and ornate text overlays into polished final compositions for the Social Marketer ecosystem.

## Target Output
- **Format**: MP4, 1080x1920 (9:16 vertical)
- **Duration**: 15-30 seconds
- **Style**: Stoic, minimalist, premium aesthetic
- **Content**: Wisdom quotes, philosophical thoughts, stoic principles

## Key Features
1. **RSS Integration**: Pulls content from wisdombook.life feeds (daily, wisdom, thoughts, quotes, passages)
2. **AI Background Generation**: Uses Gemini Veo 3.1 and Pika (Fal.ai) for looping 3D visuals
3. **Text-to-Speech**: Kokoro TTS (am_liam voice) for narration and CTA
4. **Dynamic Graphics**: 10 ornate border styles with daily rotation
5. **Cinematic Timing**: Black screen intro → background fade → text fade → narration start → CTA outro
6. **Background Music**: Mixed at 14% volume under narration

## Success Criteria
- Generate complete videos from RSS feed in single command
- Professional-grade output suitable for social media posting
- Fully automated pipeline requiring no manual intervention
- Reliable TTS with consistent voice quality
- Efficient caching to avoid redundant processing
